# Diary Front End
# ![Landing Page](public/imgs/diary.png)

The following MERN stack application is called Diary! Diary is an application where you can store informations however you want. You can upload photos that are important to you, so you can able to reference back whenever you want. Also, you can upload any codes that you learned and think is useful in many cases, you can save your code to your diary!
